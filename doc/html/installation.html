This tutorial describes the setup of a ready-to-use environment for programming with the IVT library under Windows and Linux. This includes the optional setup of optional additional libraries supported by the IVT, namely Qt or drivers for Firewire cameras.

The tutorial is divided into three main sections: Windows, Linux and Mac OS.
